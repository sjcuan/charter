Readme
=========================
To prepare the Front End and Back End ,

1) Clone the repository 
2) Run "npm install" in (frontend/restaurants and backend) both folders and the node module folder will be added.




Running the frontend
=========================
1) cd to the project folder(frontend/restaurants) and run "npm start" in the CMD

Running the Backend
=========================
(Install nodemon globally)
 Npm install -g nodemon
1) cd to the project folder and run "nodemon" in the CMD.
